# 🌾 Reyes Rice Supply Inventory System

A full-stack inventory and sales management system designed for a rice supply business.

## 🚀 Features
- User authentication (Login/Register)
- Dashboard analytics
- Product management (CRUD)
- Sales tracking
- Inventory monitoring
- Reports system

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
